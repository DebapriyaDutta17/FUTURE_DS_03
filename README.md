# 📊 Marketing Funnel & Conversion Performance Analysis Dashboard

## 🚀 Project Overview

This project analyzes a multi-channel marketing funnel to identify customer conversion patterns, channel effectiveness, acquisition costs, and opportunities to improve lead-to-customer conversion.

An interactive dashboard was built using **Python, Pandas, and Plotly** to visualize key business metrics, conversion drop-offs, ROI, and marketing performance across multiple channels and campaigns.

---

## 🎯 Objective

The primary objectives of this project were to:

* Analyze marketing funnel performance from Visitors → Leads → Qualified Leads → Customers
* Identify conversion drop-offs across different funnel stages
* Evaluate channel-wise customer acquisition performance
* Measure Cost per Customer (CPC) and Return on Investment (ROI)
* Generate actionable business recommendations to improve marketing effectiveness

## 📂 Dataset Information

The dataset consists of:

* **144 records**
* **12 months of marketing data (Jan–Dec 2025)**
* **6 marketing channels**
* **2 campaigns per channel**

### Marketing Channels

* Facebook Ads
* Google Ads
* Instagram
* Email Marketing
* Organic Search
* LinkedIn Ads

### Dataset Features

| Column          | Description            |
| --------------- | ---------------------- |
| Date            | Campaign date          |
| Month           | Month name             |
| Channel         | Marketing platform     |
| Campaign        | Campaign type          |
| Visitors        | Total website visitors |
| Leads           | Generated leads        |
| Qualified_Leads | Qualified prospects    |
| Customers       | Converted customers    |
| Spend           | Marketing expenditure  |
| Revenue         | Revenue generated      |
| ROI_%           | Return on investment   |

### 🏆 Highest Customer Acquisition Channel

**Organic Search**

### 📈 Best Conversion Performance

**Email Marketing**

### 💰 Lowest Customer Acquisition Cost

**Organic Search**

### 🚀 Highest Return on Investment

**Organic Search**

### ⚠️ Lowest Conversion Performance

**Instagram**

### 💸 Highest Cost Per Customer

**LinkedIn Ads**

---

## 💡 Actionable Recommendations

### 1. Increase Investment in Email Marketing

Email Marketing achieved the highest conversion rate and demonstrated strong customer engagement.

### 2. Scale Organic Search Initiatives

Expanding SEO strategies and content marketing efforts can maximize long-term customer acquisition and ROI.

### 3. Optimize Instagram Campaigns

Improving creatives, landing pages, and call-to-action strategies can reduce funnel drop-offs.

### 4. Refine LinkedIn Ads Targeting

Better audience segmentation and bidding strategies can lower acquisition costs and improve campaign performance.

### 5. Expand Facebook Remarketing Campaigns

Remarketing initiatives can help nurture leads and improve customer conversion rates.
